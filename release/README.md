# Médias hors Git

Les fonds d'écran ne sont pas stockés dans l'historique Git. L'asset `caelestia-wallpapers-v1.tar.zst` et sa somme de contrôle sont publiés avec la Release GitHub `wallpapers-v1`.

Pour reconstruire l'asset depuis la machine source :

```bash
./scripts/build-wallpaper-release.sh ~/Images/wallpapers
```
