# Fonds d'écran Caelestia v1

Cette Release contient l'archive optionnelle des 36 fonds d'écran utilisés par la configuration : une image statique et 35 vidéos.

L'installeur vérifie la somme SHA-256 publiée dans le dépôt avant toute extraction. Les médias sont fusionnés dans `~/Images/wallpapers` sans écraser les fichiers qui portent déjà le même nom.
